# Rock-Paper-Scissor-Game

A simple and fun Rock–Paper–Scissors game built using Python. Play against the computer in a best-of-N format with clean and easy-to-understand code.

## Features
- Play vs computer
- Best-of-N rounds
- Input validation
- Works on Windows, macOS, Linux

## How to Run
1. Install Python 3.x  
2. Clone the repo
3.  Run the game


## Project Files
- `game.py` — main game file  
- `README.md` — project info
